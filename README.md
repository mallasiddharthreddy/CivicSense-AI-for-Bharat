# CivicSense — AI for Bharat Hackathon (Student Track)

CivicSense is a Zero-UI, AI-powered first-mile waste verification system designed for gated communities in India.

This repository contains Kiro-generated specification and design documents submitted as part of the **AI for Bharat Hackathon – Student Track (Track 6: AI for Communities, Access & Public Impact)**.

## Files
- `requirements.md` — Functional and non-functional requirements (generated via Kiro)
- `design.md` — System design and AWS architecture (generated via Kiro)

The accompanying presentation deck is submitted separately as a PDF, using the official hackathon template.
